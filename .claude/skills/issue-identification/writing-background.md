# writing-background

## Purpose
Identify background information that interrupts the storyline to provide context for readers.

## Definition
Background information is material that interrupts the storyline to provide context. It may describe:
- Events that happened earlier (flashbacks)
- Circumstances existing when the main events occur (setting)
- Information about characters, places, or objects
- Events that will happen later (flash-forwards)
- Explanations of why something is significant

Background is NOT part of the main action sequence - it's supplementary information that helps readers understand the story better.

## Recognition Patterns

### Common Hebrew Markers
- Waw + subject fronting (e.g., "Now Hannah, she was...")
- Perfect tense verbs in past narrative context
- "Be" verbs (was, were) describing states
- Phrases like "in those days," "at that time"

### Common Greek Markers
- Particles: kai ("And"), de ("But/Now"), gar ("for")
- Periphrastic constructions (was + participle)
- Parenthetical information

### Content Types Signaling Background
1. **Character descriptions**: age, occupation, relationships, physical traits
2. **Setting information**: location, time, circumstances
3. **Explanatory notes**: customs, names, historical context
4. **Prior events**: what happened before the current scene
5. **Anticipatory notes**: what will happen later

## Examples from Published Notes

### Setting Information
> "Now it happened on a Sabbath that he was going through the grain fields" (Luke 6:1)
- Background: when/where the main event will occur

### Character Information
> "Now the boy was a boy" (1 Samuel 1:24) - Samuel was still very young
> "Now Eglon" was a very fat man (Judges 3:17)

### Circumstantial Background
> "Now darkness had already happened, but Jesus had not yet come to them" (John 6:17)
- Sets up the miracle that follows

### Explanatory Background
> "Now the name of Debir previously was Kiriath Sepher" (Judges 1:11)
> "in the month of Ziv (it is the second month)" (1 Kings 6:1)

### Historical Background
> "Now Laban had gone to shear his sheep" (Genesis 31:19)
- Explains why Jacob could flee without Laban knowing

## NOT This Issue

| Use Instead | When |
|-------------|------|
| grammar-connect-time-background | Ongoing background CLAUSE providing time frame for main event (while, when + ongoing action) |
| writing-newevent | Introducing a new episode (not explaining it) |
| figs-explicit | Reader needs information not stated in text |
| writing-participants | Introducing a new character (not describing them) |

## Key Distinction: writing-background vs grammar-connect-time-background

| writing-background | grammar-connect-time-background |
|-------------------|--------------------------------|
| General explanatory information | Specific ongoing action as time frame |
| Can be multiple sentences/verses | Typically a single clause |
| Explains context, history, setting | Describes what was happening WHEN main event occurred |
| "Now Eglon was a very fat man" | "While he was sitting in the upper room..." |
| "For there is no man who does not sin" | "When Solomon was old, he also worshiped..." |

## Recognition Process

```
Is the author pausing the main storyline?
  |
  +-- YES: Is it describing an ongoing action as time frame for another event?
  |         |
  |         +-- YES --> grammar-connect-time-background
  |         +-- NO: Is it explanatory/contextual information?
  |                   |
  |                   +-- YES --> writing-background
  |                   +-- NO --> Consider writing-newevent or other issues
  |
  +-- NO --> Not a background issue
```

## Common Note Patterns

1. **Verse-level background**: "This verse provides background information about the setting of the story."

2. **Multi-verse background**: "Verses 1-4 give background information about the setting of the story."

3. **Character background**: "The author is providing background information about [CHARACTER] to help readers understand what happens next."

4. **Explanatory background**: "The author is providing background information to help readers recognize that [EXPLANATION]."

5. **Circumstantial**: "Here the author is providing background information about when this event happened."

## Languages Mark Background Differently

Background information may be marked by:
- Different verb forms or tenses
- Special conjunctions or particles
- Word order changes
- Intonation patterns (in oral languages)
- Parenthetical markers

The note should remind translators to use whatever method is natural in their language.
