# writing-newevent

## Purpose
Identify phrases and constructions that introduce a new event or episode in a narrative.

## Definition
New event markers are words, phrases, or constructions that signal the beginning of a new event, scene, or episode in a story. Authors use these markers to:
- Introduce the next event in sequence
- Mark scene transitions
- Begin a new section or episode
- Indicate that time has passed since the previous event

These markers help readers understand story structure. Different languages have their own conventions for introducing new events.

## Recognition Patterns

### Hebrew Markers
- **wayyehi** ("And it happened") - Most common new event marker
- **wehayah** ("And it happened") - Variant form
- **wehinneh** ("And behold") - Draws attention to new development
- **wayyomer/wayyabo** ("And he said/came") at story beginnings
- Time phrases: "in those days," "at that time," "after these things"
- **miqetz yamim** ("at the end of days/time") - After passage of time

### Greek Markers (NT)
- **kai egeneto** ("And it happened that")
- **egeneto de** ("And it happened that")
- **meta tauta/touto** ("After these things/this")
- **de** ("Now/But") - Often introduces next major event
- **tote** ("Then") - Next event in sequence
- **kai euthys** ("And immediately") - Quick succession
- Time references introducing episodes

### Common Patterns
1. **Sequential marker + action**: "And it happened that he went..."
2. **Time reference + event**: "In those days John came preaching..."
3. **Scene setting**: New location, time, or participants introduced
4. **Connector alone**: "Now," "Then," "And," "But" at episode boundaries

## Examples from Published Notes

### Hebrew wayyehi Pattern
> "And it happened in the course of days" (Gen 4:3)
- Introduces Cain and Abel's offerings

> "And it happened at the end of forty days" (Gen 8:6)
- New development: Noah opens the window

> "And it happened" (1 Sam 1:12)
- This phrase introduces new events throughout 1 Samuel

### Greek de/kai Pattern
> "Now" (Matt 2:1) - Introduces wise men story
> "Now" (Matt 3:1) - Introduces John the Baptist
> "Then" (Matt 3:13) - Jesus comes for baptism

### Time Reference + Event
> "In the days of Herod, king of Judea" (Luke 1:5)
- Sets timeframe for Zechariah story

> "After this" (John 2:12)
- New event after wedding at Cana

> "After these things" (John 5:1)
- New event after healing in Capernaum

### Scene Transitions
> "Now Moses was shepherding" (Exod 3:1)
- Brings story focus back to Moses in Midian

> "And the king went to Gibeon" (1 Kgs 3:4)
- New episode in Solomon's reign

## NOT This Issue

| Use Instead | When |
|-------------|------|
| writing-background | Providing explanatory info ABOUT an event (not introducing one) |
| writing-endofstory | Concluding a story (not beginning one) |
| writing-participants | Introducing a new CHARACTER (use writing-newevent for the event itself) |
| grammar-connect-time-background | Describing ONGOING action as time frame for main event |

## Key Distinction: writing-newevent vs writing-background

| writing-newevent | writing-background |
|------------------|-------------------|
| Marks START of a new episode | Provides context during/before events |
| Introduces what will happen | Explains setting, characters, circumstances |
| "And it happened that..." | "Now Eli was very old..." |
| "After these things Jesus went..." | "For there was no room in the inn" |
| Sequential - moves story forward | Parenthetical - pauses storyline |

## Recognition Process

```
Is the author marking a transition to a new event/episode?
  |
  +-- YES: Is it introducing what happens NEXT?
  |         |
  |         +-- YES: Does it use wayyehi/egeneto/time phrase/connector?
  |         |         |
  |         |         +-- YES --> writing-newevent
  |         |         +-- NO: Is it still clearly a scene transition?
  |         |                   |
  |         |                   +-- YES --> writing-newevent
  |         |                   +-- NO --> Consider other issues
  |         |
  |         +-- NO: Is it providing background information?
  |                   --> writing-background
  |
  +-- NO: Is it concluding an episode?
            --> writing-endofstory
```

## Common Note Patterns

1. **wayyehi pattern**: "The author is using the phrase **And it happened** to introduce a new event in the story. Use a word, phrase, or other method in your language that is natural for introducing a new event."

2. **Greek de/kai**: "Here, the word **Now** introduces the next major event in the story. If it would be helpful in your language, you could use a word or phrase that introduces the next event, or you could leave **Now** untranslated."

3. **Time reference**: "This time reference introduces a new event."

4. **After these things**: "This phrase introduces a new event that happened some time after the events the story has just related. The story does not say how long after those events this new event happened. Use the natural form in your language for introducing a new event."

5. **Cross-reference style**: "See how you translated this phrase in [1:12](../01/12.md)."

## Languages Mark New Events Differently

New events may be marked by:
- Specific transitional words or phrases
- Changes in verb tense or aspect
- Discourse particles
- Word order changes
- Explicit time references
- Paragraph markers

The note should remind translators to use whatever method is natural in their language for introducing new events.
