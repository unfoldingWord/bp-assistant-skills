# grammar-connect-logic-result

## Purpose
Identify reason-and-result relationships where the result is stated before the reason, which may require reordering in some languages.

---

## WHEN TO WRITE A NOTE

**IMPORTANT**: We do NOT write notes for every reason-result relationship. Only write a note when:

**The REASON comes AFTER the RESULT**

This is the linguistic issue - some languages require the reason to come first. When English states the result first and explains the reason afterward, translators may need to reorder.

**Write a note for**:
- "The Jews were amazed, **because** Saul had changed" (result first, then reason)
- "Blessed are the poor, **for** yours is the kingdom" (result first, then reason)
- "Do not fear, **for** you have borne a son" (command/result first, then reason)

**Do NOT write a note for**:
- Reason-result relationships where the order is already reason-first
- Cases where the connection is clear and natural

---

## Definition
A Reason-and-Result Relationship is a logical relationship in which one event is the **reason** or cause for another event. The second event is the **result** of the first event.

From Translation Academy:
> A reason-and-result relationship is a logical relationship in which one event is the **reason** or cause for another event. The second event, then, is the **result** of the first event.

---

## CRITICAL DISTINCTION: grammar-connect-logic-result vs grammar-connect-logic-goal

These two issues use similar connecting words but have different meanings:

| Issue | Relationship | Key Test |
|-------|--------------|----------|
| **grammar-connect-logic-result** | CAUSE-EFFECT - no intention required | Is this just a consequence happening? |
| **grammar-connect-logic-goal** | PURPOSE - someone intends the outcome | Does the agent INTEND for this to happen? |

### The Same Word Can Signal Either

The phrase "so that" can indicate EITHER result or goal:

**Result (no intention)**:
> "Behold, a great storm arose on the sea, **so that** the boat was covered with the waves" (Matthew 8:24)
> - The sea cannot think - this is cause-and-effect, not purpose

**Goal (intention present)**:
> "Moses wants God to show him God's ways **so that** he may know God" (Exodus 33:13)
> - Moses INTENDS to know God - this is his purpose

**Key insight from Translation Academy**:
> Notice that the term "so that" often indicates a goal relationship, but here the relationship is reason-and-result. **This is because the sea cannot think and therefore does not have a goal.**

## Common Markers for Result Relationships

| Type | Marker | Function |
|------|--------|----------|
| Reason | **because** | Most common reason marker |
| Reason | **for** | Often introduces explanatory reasons |
| Reason | **since** | Indicates known reason |
| Result | **so** | Common result marker |
| Result | **therefore** | Formal logical conclusion |
| Result | **so that** | Result when no intention is implied |
| Result | **as a result** | Explicit result marker |
| Result | **this is why** | Explains reason for result |

---

## Two Directions of Reason-Result

| Direction | Pattern | Example | Analysis |
|-----------|---------|---------|----------|
| **Looking Backward** (most common - write note) | RESULT ... because/for ... REASON | "The Jews were amazed, **because** Saul had changed" | Result: amazement / Reason: transformation |
| | | "Blessed are the poor, **for** yours is the kingdom" | Result: blessed / Reason: kingdom |
| **Looking Forward** (no note needed) | REASON ... so/therefore ... RESULT | "A great storm arose, **so that** the boat was covered" | Reason: storm / Result: covered |
| | | "They did not consult God, **therefore** they were deceived" | Reason: no consultation / Result: deception |

---

## Categories of Result Relationships

| Category | Description | Example | Analysis |
|----------|-------------|---------|----------|
| **Explanatory "For"/"Because"** | Reason explains why statement is true | Romans 1:16: "I am not ashamed, **for** it is the power of God" | Why not ashamed? Gospel is powerful |
| **Consequential "Therefore"/"So"** | Logical conclusion follows | Romans 1:24: "**Therefore** God gave them over" | Consequence of rejecting God |
| **Natural Cause-Effect** | Physical events, no agent intention | Matthew 8:24: "A great storm arose, **so that** the boat was covered" | Storm causes waves |
| **Reason for Commands** | Why someone should do something | 1 Samuel 4:20: "Do not fear, **for** you have borne a son" | Reason not to fear: she has a son |

---

## Why This Is a Translation Issue

From Translation Academy:
> A reason-and-result relationship can look forward - "I did Y because I wanted X to happen." But usually it is looking backward - "X happened, and so I did Y." Also, it is possible to state the reason either before or after the result. Many languages have a preferred order for the reason and the result.

Translation challenges:
1. **Word order**: Some languages require reason before or after result
2. **Connector ambiguity**: Same words (like "for") can be reason or purpose
3. **Implicit connections**: Sometimes the logical connection isn't marked

## NOT grammar-connect-logic-result (Use These Instead)

### Use grammar-connect-logic-goal for:
| Expression | Why NOT result |
|------------|----------------|
| "Christ died **so that** we might be saved" | Christ INTENDED salvation - this is purpose |
| "Paul was set apart **for** the gospel" | Purpose of being set apart |
| "Keep the commandments **to** enter life" | Goal/purpose relationship |

### Use grammar-connect-condition-hypothetical for:
| Expression | Why different |
|------------|---------------|
| "**If** this happens, **then** that will happen" | Conditional, not cause-effect already happening |

## Recognition Process

1. **Identify the logical connection**: Is one event causing or explaining another?

2. **Check for intention**:
   - Is this just cause-and-effect (no agent intending)?
   - Is this explaining WHY something happened (reason)?
   - If YES --> potentially grammar-connect-logic-result
   - If an agent INTENDS the outcome --> grammar-connect-logic-goal

3. **Check the order - THIS IS KEY**:
   - Is the RESULT stated first, then the REASON?
   - If YES --> Write a grammar-connect-logic-result note
   - If NO (reason comes first) --> No note needed

4. **Write note suggesting reorder**:
   - Note should suggest reversing the clauses so reason precedes result

## Summary: Quick Decision Guide

```
Is there a logical connection between two events?
  |
  +-- Is one event CAUSING or EXPLAINING another?
        |
        +-- YES: Is an agent intentionally causing the outcome?
        |     |
        |     +-- YES (intended outcome) --> grammar-connect-logic-goal
        |     |
        |     +-- NO (just cause/effect):
        |           |
        |           +-- Is RESULT stated before REASON?
        |                 |
        |                 +-- YES --> Write grammar-connect-logic-result note
        |                 +-- NO --> No note needed (order is fine)
        |
        +-- Is it a condition ("if X then Y")?
              |
              +-- YES --> grammar-connect-condition-* issues
```

**Remember**:
1. The test for goal vs result is whether INTENTION is involved
2. For result, only write a note when RESULT comes before REASON (the reordering issue)

---

## Special Case: Non-Thinking Agents

When the "cause" cannot think or intend, it's ALWAYS result, not goal:
- Storms, weather, natural events
- Abstract concepts being personified (unless genuine personification)
- Circumstances that lead to outcomes

Example: "The storm arose **so that** the boat was covered" - The storm cannot intend to cover the boat, so this is result despite "so that."
