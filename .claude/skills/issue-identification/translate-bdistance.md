# translate-bdistance

## Purpose
Identify biblical distance/length measurements that may need translation notes.

## Definition
Biblical distance units were based on body parts (hand, arm) or standardized lengths. The metric values are approximate - biblical measures varied by time and place.

## Common Distance Units

| Unit | Description | Approximate Metric |
|------|-------------|-------------------|
| handbreadth | Width of palm | 8 cm |
| span | Spread hand width | 23 cm |
| cubit | Elbow to fingertip | 46 cm |
| "long" cubit | Cubit + span (Ezekiel 40-48 only) | 54 cm |
| stadium/stadia | Greek footrace length | 185 m |

## When to Write Notes

Write notes when these distance terms appear:
- **cubits** - Most common OT distance unit
- **span** / **handbreadth** - Smaller measurements
- **stadia** / **stadium** - NT distance unit (Greek)
- **long cubit** - Only in Ezekiel's temple vision

## NOT translate-bdistance

| Use This Instead | When |
|------------------|------|
| translate-bweight | Shekels, talents (weight of metal) |
| translate-bvolume | Ephah, hin, bath (liquid/dry capacity) |
| translate-bmoney | Coins or currency value |

## Translation Principles (from TA)

1. Original measures help readers know the Bible was written long ago
2. Modern measures help readers understand more easily
3. If possible, mention both kinds in text or footnote
4. Avoid false precision (say "half a meter" not "46 centimeters")
5. Use "about" for general measurements, but NOT when God specifies exact requirements
6. When God gives specifications for construction, do not use "about"

## Recognition Patterns

Look for:
- "cubit(s)" - arm-length measurement
- "span" - hand-spread measurement
- "handbreadth" - palm-width measurement
- "stadia" / "stadium" - NT distance unit (Greek/Roman)

## Note Approaches

### Option 1: Keep Biblical Term
Transliterate the term ("kubit") - helps readers know this is ancient

### Option 2: Use Modern Equivalent
Convert to metric - helps readers understand size

### Option 3: Provide Both
"two cubits (about one meter)" - gives both advantages

## Key Distinction: God's Specifications

**From TA**: When God commands specific measurements (tabernacle, ark, temple), do NOT say "about" because it suggests God did not care about exact dimensions.

**Approximate OK**: "Emmaus was about 60 stadia (ten kilometers) from Jerusalem" (Luke 24:13)

**Exact Required**: "Its length must be two and a half cubits" (Exodus 25:10) - God's command for the ark

## Examples from Published Notes

**Exodus 25:17**: "two cubits and a half" - measurement for mercy seat
- Notes provide metric conversion: ~115 cm

**Genesis 7:20**: "The waters rose fifteen cubits above them"
- Notes provide modern equivalent: ~22 feet

**John 11:18**: "15 stadia away"
- Notes convert to modern: ~2 miles

**1 Kings 7:26**: "a handbreadth"
- Notes explain: ~8 cm or ~3.5 inches

## Categories

### Construction Measurements
- Tabernacle/temple dimensions (exact - God's commands)
- Ark of the covenant measurements (exact)
- Other furniture specifications (exact)

### Geographic Distances
- Distance between places (often approximate)
- Depth/height of water (contextual)

### Object Sizes
- Weapons, tools, furnishings
- Usually can be approximate unless specifically commanded
