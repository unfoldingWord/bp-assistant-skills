# writing-participants

## Purpose
Identify phrases that introduce or reintroduce participants (characters) in a narrative.

## Definition
Participants are the people, animals, or beings in a story. When they first appear, they are **new participants**; when referenced again, they are **old participants**. Different languages have different conventions for:
- Introducing someone new to the story
- Reintroducing someone after they've been absent
- Referring back to established characters

This issue addresses how authors signal participant status and how translators should handle these signals naturally in their language.

## Recognition Patterns

### New Participant Introduction
Common constructions for introducing new participants:

**Hebrew patterns:**
- "And there was a man" (wayehi ish) - existential clause
- "a man from [place]" - indefinite with origin
- "whose name was X" - name introduction formula
- Relational introduction: "his wife," "his son"

**Greek patterns:**
- "There was a certain [noun]" (en tis + noun)
- "a man whose name was X"
- Indefinite article + noun: "a priest," "a woman"
- "certain X" (tis) - introduces specific but unnamed individual

**Content markers:**
- Background information: occupation, location, family
- Relationships: "wife of," "son of," "servant of"
- Descriptive traits: "who was paralyzed," "who was a leper"

### Old Participant Reintroduction
Characters return to focus after absence:
- Name + descriptor: "the man, the Levite"
- Relationship reminder: "Abigail, the wife of Nabal"
- Title restoration: "Abiathar the priest, the son of Ahimelech"
- Context connector: "the parents" (when obvious from context)

## Examples from Published Notes

### "There was a man/woman" Pattern
> "there was a certain priest" (Luke 1:5)
> "there was a man in Jerusalem whose name was Simeon" (Luke 2:25)
> "there was one man from Zorah" (Judges 13:2)
> "a certain man was sick, Lazarus from Bethany" (John 11:1)

### Indefinite + Action Pattern
> "a leper comes to him" (Mark 1:40)
> "men carrying on a mat a man who was paralyzed" (Luke 5:18)
> "a certain ruler asked him" (Luke 18:18)

### Relational Introduction
> "to a man whose name was Joseph" (Luke 1:27)
> "his wife was barren" (Judges 13:2)
> "And Gideon, his son" (Judges 6:11)

### Named Introduction
> "John came" (Mark 1:4) - assumes readers know who John is
> "Caesar Augustus" (Luke 2:1) - title + name for historical figure
> "Herodias, the wife of his brother Philip" (Matthew 14:3)

### Parable Characters
> "A certain man" (Luke 10:30, 14:16, 16:1, 19:12)
> "Two men" (Luke 18:10)
> "the sower went out" (Matthew 13:3, Mark 4:3)

### Reintroduction After Absence
> "Joshua" (Exodus 32:17) - last mentioned in 24:13
> "Jethro" (Exodus 4:18) - reintroduced as participant
> "the Pharisees who were present" (Luke 16:14) - reminds they were there

## NOT This Issue

| Use Instead | When |
|-------------|------|
| writing-background | Providing INFORMATION about a character (not introducing them) |
| writing-newevent | Introducing a new EVENT (not a character) |
| writing-pronouns | Clarifying which existing character a pronoun refers to |
| figs-explicit | Reader needs implicit information made explicit (not about narrative introduction) |

## Key Distinction: writing-participants vs writing-background

| writing-participants | writing-background |
|---------------------|-------------------|
| INTRODUCES who the character is | DESCRIBES character's traits/history |
| "There was a man named Nabal" | "Now Nabal was a harsh man" |
| Brings character into the story | Provides info about existing character |
| First mention or re-establishment | Explanatory pause in storyline |

## Recognition Process

```
Is the author marking a character's presence in the story?
  |
  +-- YES: Is this character's FIRST appearance in this narrative?
  |         |
  |         +-- YES: Does text use introduction formula?
  |         |         |
  |         |         +-- "There was a [person]" --> writing-participants
  |         |         +-- "a certain [person]" --> writing-participants
  |         |         +-- Indefinite description + action --> writing-participants
  |         |         +-- Name alone (assumed known) --> writing-participants
  |         |
  |         +-- NO: Has character been absent for significant portion?
  |                   |
  |                   +-- YES: Is author using name/description to remind readers?
  |                   |         --> writing-participants (reintroduction)
  |                   +-- NO: Is pronoun reference unclear?
  |                             --> writing-pronouns
  |
  +-- NO: Is author providing information ABOUT character?
            --> writing-background
```

## Common Note Patterns

1. **New participant**: "This phrase introduces a new character in a story. If your language has an expression of its own that serves this purpose, you could use it here."

2. **Reintroduction**: "The author is using this phrase to reintroduce [CHARACTER] as a participant. Your language may have its own way of reintroducing participants."

3. **Parable character**: "This introduces a character in the parable. Alternate translation: [There was a man who...]"

4. **Assumed knowledge**: "The author assumes readers know who [NAME] is, so there is no need to explain who he is."

5. **Relational intro**: "The author introduces [CHARACTER] by her relationship to [OTHER]. If your language has its own way of introducing new participants, you can use it here."

## Languages Introduce Participants Differently

Different languages may use:
- Existential clauses ("There was...")
- Demonstrative markers
- Specific particles for new vs known information
- Word order changes (fronting for new info)
- Special verb forms
- Definite/indefinite article distinctions (where available)
- Topic markers

The note should remind translators to use whatever method is natural in their language for introducing or reintroducing participants.
