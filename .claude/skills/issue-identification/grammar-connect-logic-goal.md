# grammar-connect-logic-goal

## Purpose
Identify logical connections where one event is done with the intent that a second event (the goal/purpose) will occur.

## Definition
A Goal (Purpose) Relationship is a logical relationship in which the second event is the purpose or goal of the first event. **Someone must do the first event with the intention that it will cause the second event.**

From Translation Academy:
> A Goal Relationship is a logical relationship in which the second event is the purpose or goal of the first event. In order for something to be a goal relationship, someone must do the first event with the intention that it will cause the second event.

---

## CRITICAL DISTINCTION: grammar-connect-logic-goal vs grammar-connect-logic-result

These two issues use similar connecting words but have different meanings:

| Issue | Relationship | Key Test |
|-------|--------------|----------|
| **grammar-connect-logic-goal** | PURPOSE - someone intends the outcome | Does the agent INTEND for the second event to happen? |
| **grammar-connect-logic-result** | CAUSE-EFFECT - no intention implied | Is this just a consequence, not an intended outcome? |

### The Same Word Can Signal Either

The phrase "so that" can indicate EITHER goal or result depending on context:

**Goal (intention present)**:
> "Moses wants God to show him God's ways **so that** he may know God" (Exodus 33:13)
> - Moses INTENDS to know God - this is his purpose

**Result (no intention)**:
> "Behold, a great storm arose on the sea, **so that** the boat was covered with the waves" (Matthew 8:24)
> - The sea cannot think and therefore does not have a goal - this is just cause-effect

## Common Markers for Goal Relationships

### English Markers
- **in order to / in order that** - Most explicit goal marker
- **so that** - Goal when intention is present
- **for** - Often introduces purpose clauses
- **to** (infinitive) - Often marks purpose
- **lest** - Negative goal (avoiding an outcome)

### Pattern Recognition
Look for these structures:
1. Action + "in order to" + intended outcome
2. Action + "so that" + intended outcome (with intentional agent)
3. Action + "for" + purpose phrase
4. Action + infinitive expressing purpose

---

## Categories of Goal Relationships

### 1. Purpose Clauses with Explicit Markers
Clear purpose markers like "in order to," "so that," "in order that"

**Examples**:
- Romans 1:1: "set apart **for** the gospel of God"
  - Purpose: Paul was set apart for the purpose of preaching the gospel
- Galatians 1:4: "gave himself for our sins **so that** he might rescue us"
  - Purpose: Christ's goal was to rescue us

### 2. Infinitive Purpose Clauses
The word "to" marks purpose when action is intentional

**Examples**:
- Leviticus 1:4: "**in order to** make atonement for him"
- Leviticus 8:11: "sprinkled them **to** consecrate them"
  - The sprinkling was done with the purpose of consecration

### 3. Negative Purpose (Lest)
Action taken to AVOID an unwanted outcome

**Template**: "SPEAKER is using the word translated as **lest** to introduce an outcome that he does not want to happen. Because of this, he first gives PEOPLE an instruction to follow so that this will not happen."

**Examples**:
- "Do not go near, **lest** you die"
  - The warning is given with the PURPOSE of preventing death

### 4. Goal of Divine Action
God acts with specific purposes/intentions

**Examples**:
- Romans 3:25: "God presented him as a propitiation **for** a demonstration of his righteousness"
  - God's purpose was to demonstrate his righteousness
- Galatians 4:5: "God sent forth his Son **in order that** he might redeem those under the law"

---

## Why This Is a Translation Issue

From Translation Academy:
> In Scripture, the goal or purpose may be stated either first or second. But in some languages, the goal or purpose must always occur in the same position (either first or second) in order for that logical relationship to be understood.

Translation challenges:
1. **Word order**: Some languages require purpose to come before or after the action
2. **Connector choice**: Languages have different ways to mark purpose
3. **Ambiguous markers**: Words like "so that" can be unclear

---

## Decision from Issues Resolved (Apr 16, 2025)

> For grammar-connect-logic-goal, we will not suggest reversing the order of clauses so that the purpose precedes the action. Instead, we will encourage translators to use a connecting word that makes it clear that the second clause describes the purpose for the first clause.

**Key Point**: Do NOT suggest reordering. Focus on clear connectors.

## NOT grammar-connect-logic-goal (Use These Instead)

### Use grammar-connect-logic-result for:
| Expression | Why NOT goal |
|------------|--------------|
| "A great storm arose, **so that** the boat was covered" | Sea has no intention - just cause/effect |
| "God blessed the seventh day **because** he rested" | Result explains reason, not purpose |
| "The Jews were amazed **because** Saul changed" | Amazement is a result, not someone's purpose |

### Use grammar-connect-condition-hypothetical for:
| Expression | Why different |
|------------|---------------|
| "**If** you do this, you will live" | Conditional relationship, not goal |

## Recognition Process

1. **Identify the logical connection**: Is there a relationship between two events/clauses?

2. **Check for intention**:
   - Can the agent of the first action INTEND the second to happen?
   - Is the second event a DESIRED outcome?
   - If YES to both --> grammar-connect-logic-goal
   - If NO (just cause/effect) --> grammar-connect-logic-result

3. **Check the connector**:
   - Is it "in order to/that," "so that," "for," "to," "lest"?
   - Does context confirm purpose/intention?

4. **Write note if needed**:
   - Is the connector unclear or missing?
   - Might readers misunderstand the purpose relationship?
