# translate-bvolume

## Purpose
Identify biblical volume/capacity measurements that may need translation notes.

## Definition
Biblical volume units measured the capacity of containers for both dry goods (grain) and liquids (wine, oil). Values varied by time and place; metric equivalents are approximate.

## Common Volume Units

### Dry Measures
| Unit | Approximate Liters |
|------|-------------------|
| omer | 2 L |
| seah | 7.7 L |
| ephah | 22 L |
| lethek | 115 L |
| homer/cor | 220 L |

### Liquid Measures

## When to Write Notes

Write notes when these terms appear:
- **ephah** - Common dry measure (~22 L)
- **homer** / **cor** - Large dry measure (~220 L)
- **omer** - Small dry measure (~2 L)
- **seah** - Medium dry measure (~7.7 L)
- **bath** - Liquid measure (~22 L)
- **hin** - Liquid measure (~3.7 L)
- **log** - Small liquid measure (~0.3 L)
- **metrete** - Greek liquid measure (~40 L)
- **measures** (when unit is implied but not stated)

## NOT translate-bvolume

| Use This Instead | When |
|------------------|------|
| translate-bweight | Shekels, talents (weight) |
| translate-bdistance | Cubits, spans (distance) |
| translate-bmoney | Coins or currency value |

## Translation Principles (from TA)

1. Original measures show the Bible was written in ancient times
2. Modern measures help readers understand quantities
3. If possible, mention both in text or footnote
4. Avoid false precision (say "about 22 liters" not "22.8 liters")
5. Use "about" for general measurements
6. Do NOT use "about" when God specifies exact amounts for offerings/obedience

## Recognition Patterns

Look for:
- Named units: ephah, homer, cor, bath, hin, log, seah, omer
- "tenth of an ephah" (common offering amount)
- "measures" when unit is implied (Hebrew sometimes omits unit name)

## Key Distinction: Exact vs. Approximate

**From TA**: When God commands specific amounts, do NOT say "about."

**Approximate OK**: "When you came for 20 measures of grain, there were only ten" (Haggai 2:16)

**Exact Required**: "a tenth of an ephah of flour" for offerings (exact obedience to God's command)

## Note Approaches

### Option 1: Keep Biblical Term
Transliterate ("bat," "hin") - shows ancient context

### Option 2: Use Modern Equivalent
Convert to liters - helps readers understand quantity

### Option 3: Use Common Container
"basket" for grain, "jar" for wine - relatable

### Option 4: Provide Both
"one bath (about 22 liters)" - gives both advantages

## Handling Implied Units

When Hebrew gives only a number without naming the unit:

**From TA**: Options include:
1. Translate literally with just the number
2. Use generic word like "measure" or "amount"
3. Use appropriate container name ("basket," "jar")
4. Use a standard unit from your translation

Example: "20 measures of grain" or "20 baskets of grain"

## Examples from Published Notes

**Leviticus 5:11**: "a tenth of an ephah of flour"
- Notes explain: ~2 liters

**1 Kings 7:26**: "contained 2,000 baths"
- Notes convert: each bath ~22 liters

**Luke 16:6**: "100 baths"
- Notes explain: ~3,000 liters or 800 gallons

**Isaiah 5:10**: "one bath... one homer... one ephah"
- Notes explain relative sizes for contrast

## Categories

### Small Amounts (personal/daily use)
- log (~0.3 L) - smallest unit
- omer (~2 L) - daily manna portion

### Medium Amounts (household/offerings)
- hin (~3.7 L) - common in sacrificial instructions
- seah (~7.7 L) - cooking/baking amount

### Large Amounts (storage/trade)
- ephah/bath (~22 L) - standard commercial unit
- homer/cor (~220 L) - bulk quantities

### Dry vs. Liquid Context
- ephah, homer, seah, omer = grain, flour
- bath, hin, log = wine, oil, water
