# grammar-connect-time-background

## Purpose
Identify background clauses describing an ongoing situation that provides the time frame for main events.

## Definition
A background clause describes something ongoing. Another clause then indicates an event that begins during that time. The ongoing event serves as the background/context for the main event. The background clause provides the time frame, not just general context.

From Translation Academy:
> "A background clause is one that describes something that is ongoing. Then, in the same sentence, another clause indicates an event that begins to happen during that time."

## Recognition Patterns

### Structural Pattern
```
BACKGROUND CLAUSE (ongoing) + MAIN EVENT (punctual)
```

The background is what WAS happening; the main event is what BEGAN to happen.

### Common Markers
**English connectors**: "when," "while," "during," "as," "in those days," "at that time"

**Hebrew patterns**:
- Waw + subject + participle ("Now he was sitting...")
- Temporal phrases + ongoing state ("in those days... there was no king")

**Greek patterns**:
- Genitive absolute constructions
- en to + infinitive ("while they were...")
- Participles describing ongoing action

### Key Feature
The background describes something ALREADY IN PROGRESS when the main event starts.

## Examples from Published Notes

### "While" Introducing Background
> "And it came about that, **while** they were there, the days were fulfilled for her to give birth." (Luke 2:6)
- Background: being in Bethlehem (ongoing)
- Main event: birth (begins)

### "When" Introducing Background
> "**When** Solomon was old, he also worshiped their gods." (OBS Story 18)
- Background: being old (ongoing state)
- Main event: worshiping other gods (begins)

### Multiple Background Clauses
> "In the fifteenth year of the reign of Tiberius Caesar--**while** Pontius Pilate was governor of Judea, and Herod was tetrarch of Galilee... **during** the high priesthood of Annas and Caiaphas--the word of God came to John." (Luke 3:1-2)
- Background: five ongoing political/religious situations
- Main event: God's word comes to John

### Ongoing State as Context
> "And the people were expecting... Now the people were expecting" (Luke 3:15)
- Background: expectation (ongoing)
- Main event: John's response (follows)

### Time Frame Phrase
> "And his parents went **every year** to Jerusalem... And when he was 12 years old, they went up" (Luke 2:41-42)
- Background: annual custom (ongoing/repeated)
- Main event: specific trip when Jesus was 12

## NOT This Issue

| Use Instead | When |
|-------------|------|
| writing-background | General explanatory information, not specifically a time-frame clause |
| grammar-connect-time-simultaneous | Two events happening at the same time with equal weight |
| grammar-connect-time-sequential | Events happening one after another |
| figs-explicit | Adding implied information not stated in text |

## Key Distinction: background vs simultaneous

| grammar-connect-time-background | grammar-connect-time-simultaneous |
|--------------------------------|----------------------------------|
| One event is ongoing context | Both events have equal importance |
| One event is the main focus | Both events are focal |
| Background just sets the stage | Both events matter narratively |
| "While he was sleeping, she entered" | "They were eating and drinking" |

## Recognition Process

```
Does text describe an ongoing situation?
  |
  +-- YES: Does another event BEGIN during this ongoing situation?
  |         |
  |         +-- YES: Is the ongoing situation providing TIME FRAME for the event?
  |         |         |
  |         |         +-- YES --> grammar-connect-time-background
  |         |         +-- NO: Is it general explanatory context? --> writing-background
  |         |
  |         +-- NO: Are both events simultaneous with equal weight?
  |                   --> grammar-connect-time-simultaneous
  |
  +-- NO --> Not a background time issue
```

## Common Note Patterns

1. **Connector clarification**: "Luke uses the word **And** to introduce background information that will help readers understand what happens next. Alternate translation: [Now]"

2. **Background clause identification**: "The author is using **CLAUSE** to describe something that was ongoing. This provides the background for other events..."

3. **Setting the scene**: "This clause gives information about something that was happening while [MAIN EVENT]. Translate this in a way that makes that clear."

## Languages Handle Background Clauses Differently

Languages may mark background clauses through:
- Special connecting words (English: "while," "when," "as")
- Different verb forms (imperfective/progressive aspect)
- Word order changes
- Subordination markers

Notes should encourage translators to use whatever method is natural for marking background-to-main event relationships in their language.
