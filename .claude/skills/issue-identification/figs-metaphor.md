# figs-metaphor

## Purpose
Identify metaphors in biblical text, distinguishing them from other figures of speech.

## Definition
A metaphor speaks of one thing (Topic) as if it were a different thing (Image) to highlight a similarity (Idea/Point of Comparison).

## Metaphor Components
For each identified metaphor, extract:
1. **Topic**: The item being discussed by the writer/speaker
2. **Image**: The physical item (object, event, action) used to describe the topic
3. **Idea**: The abstract concept or quality connecting Topic and Image (often implied)

## Active vs Passive Metaphors

### Passive (Dead) Metaphors
Common expressions no longer recognized as figurative. Translate meaning directly without drawing attention to the figure.

### Active Metaphors
Recognized figurative speech that draws attention. Consider preserving the image or offering simile alternative.

| Pattern | Classification | Notes |
|---------|---------------|-------|

| "Behold!" (exclamation at visual) | figs-exclamations | Actually looking at something |

Default in narrative: figs-metaphor for "behold" introducing information.

| Pattern | Classification | Notes |
|---------|---------------|-------|

**Test**: Does "sons of" mean descendants/people? Use figs-metaphor.
Does "sons of" describe a characteristic? Use figs-idiom.

## NOT figs-metaphor (Use These Instead)

### Use figs-simile for:
Comparisons using explicit words "like," "as," or "than":
- "Your teeth are **like** a flock of sheep" - explicit comparison = simile
- "Sharper **than** any two-edged sword" - uses "than" = simile

**Key test**: Metaphor speaks of X AS IF IT WERE Y (identity). Simile says X is LIKE Y (explicit comparison).

### Use figs-idiom for:
| Expression | Reason |
|------------|--------|
| "in the eyes of" | Common expression, no longer active comparison |
| "from the face of" (= because of) | Causal meaning |
| "nose burns" (anger) | Cultural idiom for emotion |
| "struck to mouth of sword" | Set phrase |
| "arise/turn" (= take action) | Command to begin doing |
| "sons of X" (= characterized by X) | e.g., "sons of valor" = valiant men |
| "found" (= was there) | Characteristic usage |

### Use figs-metonymy for:
| Expression | Reason |
|------------|--------|
| "to the face" (= presence) | Association, not comparison |
| "name" (= reputation) | Associated concept |
| "days" (= time period) | Associated concept |
| "hand" (= power/control) | Associated concept |
| "word" (= message) | Associated concept |
| "ears" (= hearing) | Associated concept |
| "soul" (= person spiritually) | Associated concept |

### Use figs-synecdoche for:
| Expression | Test | Reason |
|------------|------|--------|
| "mouth of Yahweh" | Can substitute whole: "Yahweh" | Part for whole |
| Body part for person | Whole substitutes for part | Part for whole |
| "gates" (= city borders) | Physical boundary meaning | Part for whole |

**Synecdoche Test**: If the whole can substitute for the part ("they did not consult the mouth of Yahweh" = "they did not consult Yahweh"), it's synecdoche. If not ("gave into the hand" =/= "gave into"), it's metonymy.

## Recognition Process

1. **Identify potential figure**: Does the text speak of one thing as if it were another?

2. **Check Issues Resolved list**: Is this expression already classified?

3. **Apply distinction tests**:
   - Is it a set cultural phrase with fixed meaning? -> Likely **idiom**
   - Is it association (X represents Y by connection)? -> Likely **metonymy**
   - Can whole substitute for part? -> Likely **synecdoche**
   - Is it comparison (X spoken of AS IF it were Y)? -> Likely **metaphor**

4. **Determine active vs passive**:
   - Is the comparison still noticed/impactful? -> Active
   - Is it so common it's not noticed? -> Passive
