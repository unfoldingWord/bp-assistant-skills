# figs-idiom

## Purpose
Identify idioms in biblical text, distinguishing them from metaphors, metonymy, and synecdoche.

## Definition
An idiom is a group of words that, as a whole, has a meaning different from what one would understand from the meanings of the individual words. The meaning is culturally bound - someone outside the culture cannot usually understand it without explanation.

Key characteristics:
- Fixed expression with non-compositional meaning
- Cultural/linguistic convention, not active comparison
- The figure is not "noticed" as figurative - it functions as a set phrase

## NOT figs-idiom (Use These Instead)

### Use figs-metaphor for:
| Expression | Reason |
|------------|--------|
| "sons" = descendants | Active comparison (offspring imagery) |
| "turn" = change life course | Directional comparison |
| "way/path" = behavior | Journey comparison |
| "heart" = thoughts/feelings | Body-to-inner-person comparison |
| "house" = temple or descendants | Building/dwelling comparison |
| "fathers" = ancestors | Family comparison |
| "behold" = listen | When "look" actively = "listen" |

### Use figs-metonymy for:
| Expression | Reason |
|------------|--------|
| "to the face" (= presence) | Association (face seen when present) |
| "from the face" (= presence) | Association, NOT causal |
| "name" (= reputation) | Associated concept |
| "days" (= time period) | Associated concept (time made of days) |
| "hand" (= power alone) | Associated concept |
| "word" (= message) | Associated concept |
| "struck" = "defeated/killed" | Result by association with action |

### NOT requiring notes:
| Expression | Reason |
|------------|--------|
| "go up/go down" (elevation) | Hebrew marks what English doesn't - discuss in book intro only |

## Recognition Process

1. **Identify potential idiom**: Is this a set phrase whose meaning differs from literal word meanings?

2. **Check Issues Resolved list**: Is this expression already classified?

3. **Apply the idiom test**:
   - Is it a culturally-bound expression with fixed meaning?
   - Would someone outside the culture NOT understand it from word meanings alone?
   - Is it NOT an active comparison (that would be metaphor)?
   - Is it NOT association-based (that would be metonymy)?

4. **Distinguish from similar figures**:

   | Test | If YES | If NO |
   |------|--------|-------|
   | Fixed cultural phrase, non-compositional meaning? | Likely idiom | Check other figures |
   | Active comparison (X as if Y)? | Metaphor | Could be idiom |
   | Association (X represents related Y)? | Metonymy | Could be idiom |
   | Part for whole (substitutable)? | Synecdoche | Could be idiom |

5. **Key distinction: idiom vs metaphor**:
   - Idiom: Set phrase, meaning is conventional, figure not "noticed"
   - Metaphor: Active comparison, reader notices the image

## Critical Distinctions

| Expression Type | figs-idiom | Other Classification |
|-----------------|------------|---------------------|
| **Face** | "in the eyes of" (judgment), "from the face of" (causal), "face" = surface, "face to face" | "to the face of" (presence) = figs-metonymy |
| **Son** | "son of X" = characterized by X | "sons" = descendants = figs-metaphor |
| **Turn/Arise** | "turn/arise" = begin action | "turn" = change life direction = figs-metaphor |
| **Days/Day** | "day" = occasion | "days" = time period = figs-metonymy |
| **Hand** | "hand" = power + possession together | "hand" = power alone = figs-metonymy |
| **Behold** | "behold" = pay attention, "behold me" (Yahweh announcing) | "behold" = look meaning listen = figs-metaphor |
